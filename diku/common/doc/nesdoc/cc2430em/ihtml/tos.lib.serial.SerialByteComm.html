Interface: tos.lib.serial.SerialByteComm

interface SerialByteComm

A basic byte-level interface to a serial port.

Author:
David Gay
Philip Levis
Ben Greenstein
Date:
August 7 2005

Commands
command error_t put(uint8_t data) Put a single byte to the serial port.

Events
event void get(uint8_t data) Receive a single byte from the serial port.
event void putDone() Split phase event to indicate that the last put request has completed.

Commands - Details

put

command error_t put(uint8_t data)

Put a single byte to the serial port.

Parameters:
data - The byte to send to the serial port.
Returns:
Returns an error_t code indicating whether this byte was successfully put (SUCCESS) or not (FAIL).

Events - Details

get

event void get(uint8_t data)

Receive a single byte from the serial port.

Parameters:
data - The byte that has been received from the serial port.

putDone

event void putDone()

Split phase event to indicate that the last put request has completed.
